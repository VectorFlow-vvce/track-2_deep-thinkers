## Project Overview

This project trains a semantic segmentation model on synthetic desert environment images generated by **Duality AI's Falcon platform**. The model segments **11 classes** of desert terrain including trees, rocks, sky, dry grass, and more — enabling autonomous UGV (Unmanned Ground Vehicle) navigation in off-road environments.

## Model Architecture

| Component | Detail |
|-----------|--------|
| Architecture | DeepLabV3+ |
| Backbone | EfficientNet-B4 (ImageNet pretrained) |
| Input Size | 384 × 384 |
| Batch Size | 2 |
| Epochs | 40 (+ 20 resume) |
| Base LR | 3e-4 (AdamW) |
| Scheduler | CosineAnnealingWarmRestarts |
| Loss | Focal (γ=2) + Dice (40% weight) |
| Device | GPU (CUDA) / CPU fallback |
| Mixed Precision | Yes (GPU only) |

---

## Techniques Used

### Loss Function
- **Focal Loss** (γ=2): Focuses training on hard/rare examples
- **Dice Loss**: Directly optimizes IoU-like overlap metric
- **Class Weights**: Rare classes (Flowers, Logs) get 5× weight

### Augmentation Pipeline
- Horizontal/Vertical flip, RandomRotate90
- ShiftScaleRotate (scale ±10%, rotate ±15°)
- RandomBrightnessContrast, HueSaturationValue, CLAHE
- GaussNoise, GaussianBlur, MotionBlur
- CoarseDropout (random patch masking)

### Test-Time Augmentation (TTA)
- 8 augmentation variants per image
- 3 scales: 0.75×, 1.0×, 1.25×
- Weighted ensemble: original scale gets 60% weight
- Flip-back correction for flipped variants
